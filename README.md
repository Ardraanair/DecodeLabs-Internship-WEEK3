
Prompt Engineering Project 3 – Context-Anchored Answering (RAG Basics)
Overview

This project demonstrates the implementation of Context-Anchored Answering (RAG Basics) to reduce AI hallucinations by restricting responses to a provided reference document.

Key Features
Context Locking
Negative Constraints
Paragraph-Level Citations
Hallucination Prevention
Retrieval-Augmented Generation (RAG) Principles
Test Cases

The project includes:

Valid context-based queries
Out-of-context queries
Citation verification
"Information Not Found" fallback responses
